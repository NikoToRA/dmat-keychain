## 変更ログの使い方

週次レポート（毎週月曜8:30）でこのファイルを参照し、変更後の数値変化を報告する。
効果測定の観点：

1. **直帰率** — フィーチャーバー追加でコンテンツへの関与が増えたか
2. **滞在時間** — キャッチコピー変更でページ閲覧が深まったか
3. **/product 遷移率** — 「受注生産」訴求が商品詳細への誘導につながったか
4. **Organic流入** — NFC訴求タイトルでGoogle検索流入が発生し始めたか
5. **Clarityヒートマップ** — フィーチャーバーの視認・クリック状況
